# Day 41 — Attendance System (CLI)

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A command-line attendance tracker for students that
stores records in CSV and shows subject-wise stats.

## How to run
```bash
python3 attendance.py
```

## Features
- Mark attendance for multiple subjects
- View attendance percentage per subject
- Warning if attendance drops below 75%
- Export attendance report to CSV
- Stores data persistently in JSON
